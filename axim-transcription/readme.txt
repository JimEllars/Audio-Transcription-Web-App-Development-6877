Professional AI-powered transcription service widget that can be embedded anywhere using shortcode.

== Description ==

AXiM Transcription Service allows you to easily embed a professional transcription service widget on your WordPress site using a simple shortcode.

**Features:**
* Simple shortcode integration: `[axim_transcription]`
* Professional dark-themed UI
* Two pricing tiers (Student and Basic plans)
* Analytics dashboard for tracking widget performance
* Customizable default plans
* Mobile responsive design
* Integration with Supabase for analytics

**Shortcode Usage:**
* Basic: `[axim_transcription]`
* With options: `[axim_transcription default_plan="basic"]`

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/axim-transcription/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to 'AXiM Transcription' in your admin menu to configure settings
4. Use the shortcode `[axim_transcription]` on any page or post

== Frequently Asked Questions ==

= How do I embed the widget? =

Simply use the shortcode `[axim_transcription]` in any post, page, or widget area.

= Can I customize the default plan? =

Yes, you can set a default plan in the settings or use the shortcode parameter: `[axim_transcription default_plan="student"]`

= Does it work with page builders? =

Yes, the shortcode works with all major page builders including Elementor, Gutenberg, and others.
